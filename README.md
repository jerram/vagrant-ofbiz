# Ofbiz Vagrant Sandbox

## Vagrant deps

```
vagrant plugin install vagrant-hostmanager vagrant-cachier vagrant-vbguest vagrant-bindfs vagrant-env
```

## Spin up Ofbiz server:

```
vagrant up
```

open https://ofbiz.local:8443/ordermgr/control/main

- USERNAME:- admin
- PASSWWORD:- ofbiz
